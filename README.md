# 🚀 E-commerce Store

Tienda online frontend con catálogo de productos, carrito de compras y checkout. Interfaz moderna y responsive.

## 📌 Descripción
Aplicación frontend de e-commerce que permite a los usuarios navegar productos, agregarlos al carrito y realizar compras. Integración con backend API para gestión de datos.

## 🛠️ Tecnologías
- React 18
- TypeScript
- Tailwind CSS
- React Router v6
- Lucide React
- Vite

## ⚡ Features
- Catálogo de productos
- Detalle de producto
- Carrito de compras
- Checkout
- Diseño responsive
- Tema oscuro moderno

## 🌐 Demo
**https://ecommerce-store-demo.netlify.app/**

## 📦 Instalación

```bash
# Clonar el repo
git clone https://github.com/renzonodari9/ecommerce-store.git
cd ecommerce-store

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```

**Importante**: Requiere que el backend (ecommerce-api) esté corriendo en el puerto 3001.

## 📁 Estructura

```
src/
├── components/     # Componentes reutilizables
├── pages/          # Páginas principales
├── services/       # Servicios API
└── App.tsx         # Rutas y configuración
```

## 🌐 Variables de Entorno

```env
VITE_API_URL=http://localhost:3001
```
