# E-commerce Store Frontend

Tienda online para clientes con React, TypeScript y Tailwind CSS.

## Funcionalidades

- Página de inicio con productos
- Detalle de producto
- Carrito de compras
- Login/registro
- Checkout con dirección de envío
- Diseño responsive

## Instalación

```bash
npm install
npm run dev
```

Abrir http://localhost:5175
